## Zod

TypeScript-first schema validation with static type inference

## Basic usage

<Code
  terminal
  highlights={[[1,1],[4,5],[7,9],[11,13]]}
  customSetup={{
    dependencies: {
      zod: '3.19.1',
    },
  }}
  files={{
  'index.ts': `import { z } from 'zod';
import l from './log';

// Creating a schema for strings
const simpleSchema = z.string();

// Parsing
simpleSchema.parse('john');
// simpleSchema.parse(2);

// "Safe" parsing
l( simpleSchema.safeParse("tuna") );
l( simpleSchema.safeParse(12) );`,

// All properties are required by default
const DogSchema = z.object({
  name: z.string(),
  age: z.number(),
});

// Classic Type assignment
type Dog = z.infer<typeof DogSchema>;

const coopy: Dog = {
  name: 'Cooper',
  age: 2,
};
DogSchema.parse(coopy);

// Inferred Type from parsing
const marius = DogSchema.parse({
  name: 'Marius',
  age: 7
});`,

# API Validation

---

<Code
  terminal
  highlights={[[4,8],[10,15],[16,18],[20,30]]}
  customSetup={{
    dependencies: {
      zod: '3.19.1',
      miragejs: '0.1.45'
    },
  }}
  files={{
    'index.ts': `import { z } from 'zod';
import { createServer } from 'miragejs';

createServer({ routes() { this.get('/movies', () => [
  { id: 1, name: "Inception", year: 2010 },
  { id: 2, name: "Interstellar", year: 2014 },
  { id: 3, name: "Dunkirk", year: 2017 },
]); this.logging = false; }});

const moviesSchema = z.object({
  id: z.number(),
  name: z.string(),
  year: z.number()
}).array();

const getMovies = () => fetch('/movies')
  .then(res => res.json())
  .then(res => moviesSchema.safeParse(res));

(async () => {
  const movies = await getMovies();

  if (movies.success) {
    movies.data.forEach(({ name, year }) => {
      console.log(name + ' - ' + year);
    });
  } else {
    console.error(movies.error);
  }
})();`,
  }} />

<Code
  terminal
  highlights={[[2,3],[6,9],[11,13],[15,21],[23,31]]}
  customSetup={{
    dependencies: {
      zod: '3.19.1',
      miragejs: '0.1.45',
      axios: '1.1.2',
      'zod-form-data': '1.2.2'
    },
  }}
  files={{
    'index.ts': `import axios from 'axios';
import { z } from 'zod';
import { zfd } from 'zod-form-data';
import { createServer } from 'miragejs';

createServer({ routes() { this.post('/dog', (_schema, request) => {
  const { name } = Object.fromEntries(request.requestBody);
  return name + ' has been added with success!';
}); this.logging = false; }});

const formSchema = zfd.formData({
  name: zfd.text(),
  age: zfd.numeric(z.number().min(0).max(20)),
});

const setDog = (formData: FormData) => new Promise((res, rej) => {
  const validation = formSchema.safeParse(formData);
  if (validation.success) {
    axios.post('/dog', formData).then(({ data }) => res(data));
  } else { rej(validation.error); }
});

(async () => {
  document.body.innerHTML =
    '<form id="form"><input type="text" name="name" value="Cooper" />'+
    '<input type="number" name="age" value="2" /></form>';

  setDog(new FormData(document.getElementById('form')))
    .then(res => console.log(res))
    .catch(err => console.error(err));
})();`,
  }} />

## Best practices - naming

- Schema Name → <span className="text-green">`[camelCase NAME]Schema`</span> (f.ex. moviesSchema)
- Type Name → <span className="text-green">`[PascaleCase NAME]`</span> (f.ex. Movies)

---

## Patterns

- Types ? <Appear>→ Don't over use it, favor inference</Appear>
- Inference ? <Appear>→ Prefer schema.parse instead of `z.infer<>`</Appear>
- Parse location ? <Appear>→ In the service/hook</Appear>
- TS Types or Zod Schema ? <Appear>→ Zod for data, TS for the rest (args, props, etc)</Appear>
- Parse or SafeParse ? <Appear>→ SafeParse if you need the success check, but Parse in a catched error wrapper.</Appear>
